FRC 2016 LabVIEW Robot Simulation


For instructions on how to use simulation see Tutorial 10 - Robot Simulation on the Tutorials tab of the LabVIEW Getting Started Window.

Actuators on this simulated robot

  1. Left Motor
    • PWM Channel = PWM 0
  2. Right Motor
    • PWM Channel = PWM 1
  3. Camera Servo
    • PWM Channel = PWM 5
    • Angular Range = 170

Sensors on this simulated robot

  1. Encoder on Right Motor
    • A Channel = DIO 3
    • B Channel = DIO 4
  2. Encoder on Left Motor
    • A Channel = DIO 5
    • B Channel = DIO 6
  3. Gyro
    • Analog Channel = AI 1
  4. SRF04 Ultrasonic
    • Ping DIO Channel = DIO 1
    • Echo DIO Channel = DIO 2
  5. AXIS M1011 Camera
